# Calculator App

A responsive calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations
- Keyboard support
- Responsive design
- Backspace functionality
- Error handling
- Modern UI

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
